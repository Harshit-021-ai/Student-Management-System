# STUDENT-MANAGEMENT-SYSTEM
A System To Manage Student Records Like: 
                                        Name, 
                                        Roll Number, 
                                        Marks, 
                                        Attendance
Level: Beginner
Features: 
          Add student, 
          View all students, 
          Update student, 
          Delete student, 
          Save & load from file.
Concepts Used: 
              Dictionary → store each student, 
              List → store multiple students, 
              Functions → add, update, delete, search, 
              OOP → Student class, 
              File handling → save data to file, 
              Algorithms → search student by roll number
